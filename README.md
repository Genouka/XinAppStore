现在莘应用手表应用商店完全开源！

# 编译&环境

开发者使用的签名不会公开，请自行准备签名。

使用Aide Lua Pro 5.1.1版本新建项目，拷贝本项目到`app/src`文件夹中，然后点击编译运行按钮。
